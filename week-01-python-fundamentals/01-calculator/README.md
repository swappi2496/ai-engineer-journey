# Calculator CLI
A simple command-line calculator that takes two number and an operator returns the result, and handles divide by zero gracefully.

## What I learned
- Defining functions with `def`
- `input()` returns a string: convert with `float()` for math
- `if / elif / else` for rounting on the operator
- f-string for formatted output (`f"Result: {result}"`)
- The `if __name__ == "__main__":` idiom

## How to run
```bash
python calculator.py
```

## Sample run
```
First number: 12
Operator (+, -, *, /): +
Second number: 5
Result: 17.0
```
